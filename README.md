# Etern'Aile - Guide des 4 Corps

Plateforme web statique pour :

- identifier le type de difficulte vecue via l'Oracle des 4 Corps ;
- explorer la compatibilite de couple via la Resonance ;
- collecter les demandes de contact et inscriptions localement dans l'espace admin.

## Fichiers principaux

- `index.html` : page principale de la plateforme.
- `cartas/` : images des arcanes.
- `images/` : visuels de marque et fonds.

## Utilisation locale

Ouvrir `index.html` dans un navigateur ou servir le dossier avec un serveur local.

L'espace admin est accessible depuis le bouton Espace membre.
